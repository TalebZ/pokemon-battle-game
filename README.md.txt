Pokémon Battle Game

This is a simple **Pokémon battle game** built in Python using the [PokéAPI](https://pokeapi.co/).

How to Play
1. Choose your Pokémon.
2. Battle a random opponent.
3. Win by reducing your opponent's HP to 0!

Setup
1. Clone this repository:  
git clone https://github.com/TalebZ/pokemon-battle-game.git

2. Install dependencies:
pip install requests

3. Run the game:
python battle.py
or
py battle.py
depending on whether you are using PyCharm or not

- Fetches real Pokémon data
- Randomised opponents
- Simple turn-based combat
